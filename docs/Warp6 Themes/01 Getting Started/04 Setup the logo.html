Set up the logo

This tutorial will show you how to set up the logo.

Logo

The Logo is actually a simple image tag. In Joomla we are using a Custom HTML module, in Wordpress a Text widget published on the themes logo position for it to display.

Joomla

Go to Extensions � Modules Manager. Click on New. Select a Custom HTML module.

On the following edit screen, you select the logo template position.

And fill the Custom Output field. This is demonstrated below.

After publishing the module, the logo will display in the template

Wordpress

Go to Appearance � Widgets and drag & drop a Text widget to the logo position.

Edit the newly added widget and fill the field. This is demonstrated below.

Favicon

The Favicon is located in your templates root directory.

Joomla: templates/yoo_nano/favicon.png

Wordpress: wp-content/themes/yoo_nano/favicon.png

To change the favicon, simply replace that file.

iPhone Icon

The iPhone Icon is located in your templates root directory.

Joomla: templates/yoo_nano/apple_touch_icon.png

Wordpress: wp-content/themes/yoo_nano/apple_touch_icon.png

To change the iPhone Icon, simply replace that file.
